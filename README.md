## 🚀 Installation Guide

Follow these steps to set up your WordPress site:

1. **Download & Upload**: Extract the package and upload all files to your server.
2. **Run the Installer**: Open your browser and navigate to `wp-admin/install.php`.
3. **Database Setup**: Enter your database connection details when prompted.
   - If automatic setup fails:
     1. Open `wp-config-sample.php` in a text editor.
     2. Enter your database details manually.
     3. Save it as `wp-config.php` and upload it.
4. **Complete Installation**: Follow on-screen instructions and save your login credentials.
5. **Log In**: Visit `wp-login.php` and sign in.

## 🔄 Updating WordPress

### 🔹 Automatic Update
1. Go to `wp-admin/update-core.php` in your browser.
2. Follow the on-screen instructions.

### 🔹 Manual Update
1. Backup your files (especially modified ones).
2. Delete old WordPress files except `wp-config.php` and `wp-content/`.
3. Upload the new files.
4. Run `wp-admin/upgrade.php` in your browser.

## 🔄 Migrating from Other Systems

If switching from another platform, WordPress offers built-in import tools:
- Visit `wp-admin/import.php` for migration options.
- Additional details: [WordPress Import Guide](https://developer.wordpress.org/advanced-administration/wordpress/import/)
